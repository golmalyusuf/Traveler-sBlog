# Traveler-sBlog
This is a blog where travels can post there memories of their expedition.

Used Technologies: 
  Spring Boot
  Spring Security
  Hibernate
  Thymeleaf
  JPA
  Logger
  Mysql

To user this application you just need to crate a database in mysql named as "travelblog".
That's it. You are ready to go.
